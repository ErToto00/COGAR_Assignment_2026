# List of left hand gestures and their command correspondent

## X corresponds to don't mind

1. All fingers open, Thumb X, Wrist X-> STOP
2. Index and Pinky open, medium and ring down, Thumb X, Wrist X -> ROTATE ON YOURSELF
3. Pinky open, all the other fingers closed, Thumb X, Wrist X -> GO BACK
4. Index open, all the other fingers closed, Thumb X, Wrist X -> GO FORWARD
5. Thumb open, all the other fingers closed, Wrist up -> STOP, ROTATE LEFT, GO FORWARD
6. Thumb open, all the other fingers closed, Wrist down -> STOP, ROTATE RIGHT, GO FORWARD
